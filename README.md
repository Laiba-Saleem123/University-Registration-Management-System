#  University Registration Management System

A **console-based University Registration Management System** developed in **C++** using **Object-Oriented Programming (OOP)** principles.

It allows users to **add, display, search, and delete student records**, as well as manage **course registration** and **fee information** through a simple menu-driven interface.

##  Features

* Add Student Records
* Display Student Information
* Search Students by ID
* Delete Student Records
* Course Registration
* Fee Management

## Technologies

* C++
* Object-Oriented Programming (OOP)
* Dev C++

##  OOP Concepts

* Classes & Objects
* Encapsulation
* Abstraction
* Inheritance
* Polymorphism
* Dynamic Memory Allocation
